Welcome to the Angular JS Challenge for ActualFood

This app allows you to create, edit, delete, and sort through a hierarchical list

From here you can do a number of things.

  • You can create new data using the "Create Project" link in the top right corner.
  • You can edit the available departments/resources using the "Edit Categories" link in the top right corner.
  • You can select a sorting method to the side.

It appears that you do not have any data to sort.

Please create some data using the assignment tool to get started.
